Título: Viajes Nacionales

Descripción: es una aplicación (o proyecto) dedicada a gestionar, organizar y promocionar viajes dentro del territorio nacional en Venezuela.  El objetivo es ofrecer una experiencia sencilla y completa para los usuarios interesados en turismo local.

Características: * Búsqueda de destinos nacionales. * Informacion de los paquetes Turisticos. * Video del destino top del momento. * Login de la pagina.

Licencia: La licencia bajo la que se distribuye el proyecto.

Recursos que se usaron: HTML, CSS, Js y GitHub.
